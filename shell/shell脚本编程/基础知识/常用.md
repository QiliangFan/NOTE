#### 1. 从指令中获取值:

~~~shell
DATE = `date`
echo $DATE
# 使用指令或$(指令)的方式, 可以在运行时获取指令的返回值
~~~

#### 2. printf 和 echo

~~~shell
printf "%s\n" $DATE
echo $DATE # 默认自动换行
~~~

#### 3. 多重指令:

1.  ;

~~~shell
# 使用;作为一条命令的结束, 用来分割连续的多条命令.
# 当一条命令执行失败, 依旧会执行下一条指令
~~~

2. &&

~~~shell
# 使用&& 作为多条命令的分割
# 当一条命令执行失败, 不会继续执行下一条指令
~~~

3. ||

~~~shell
# 使用||作为多条命令的分割
# 当一条命令执行失败, 才执行下一条指令
~~~



#### 4. 历史记录:

~~~shell
# 1.使用上下键切换执行过的指令
# 2. 使用感叹号: 执行最近一次执行过的指令, 后面接要执行指令的开头字符
ls
sudo
!l  -> ls将会被执行
!s -> sudo将会被执行
!su -> sudo
!?su  # 包含su的, 最后执行的指令
!!   ->  重新执行最后执行的命令
!-1 -> 重新执行最后执行的命令
!-2  -> 重新执行倒数第二个已经执行的命令
sudo; !#  ->  sudo;sudo
sudo;date;!#  -> sudo;date;sudo;date 
~~~

~~~http
! 后面接指令开头字符、索引、#
!# 可以使多重指令执行两边
~~~



#### 5. cd

~~~shell
cd - # 返回到上一次目录
cd  # 返回用户主目录, 同cd ~
~~~



#### 6. 浏览文件:

~~~shell
# less, 执行完会返回到上一条指令的下面, 而more则是文件内容的下面,
#上一页: page up/b
# 下一页: page down/space
# 首页: 1G (数字1, 和大写G)
# 尾页: G
# 查找字符串: /<字符串>
# 查找下一个: n
# 查找上一个: N
# 退出: q
~~~
